# HAWC Image processing source finding pipeline using 3ML likelihood fitting  

Automated pipeline scripts for detection and analysis of gamma ray sources with a combination of the likelihood framework and the source fitting framework. The details of the source seeding framework based on image processing can be found on the Github (https://github.com/rishibbdb/WFS-Source-Detector).


## Features

- **Automated Blob Detection**: Multi-scale detection using Difference of Gaussians (DoG)
- **Logic Based Filtering**: Separates point sources (PS) from extended (EXT) sources
- **ROI Management**: Configurable Region of Interest with automatic coordinate conversion
- **YAML-based Configuration**: Easy-to-use configuration file format
- **3ML Integration**: Generates ready-to-use 3ML model files for source fitting
- **Diffuse Template Support**: Optional incorporation of diffuse background templates (e.g., HERMES data)
- **Parallel Processing**: Uses ThreadPoolExecutor for efficient multi-threaded blob detection
- **Likelihood Fitting**: Uses 3ML+HAL likelihood framework to fit the 3ML model files generated by the source seeding framework. 

**NOTE: The code is currently in alpha build**. Features such as testing different morphology and spectrum models are not currently implemented and will be introduced in beta version dev. 

## Future implementation. 
- Full analysis chain starting with finding sources based on the image processing-based seeding and generating a final model 
- Testing different morphological and spectral models
- Implement detailed logging in multiple level log files. 
- Implement a checkpoint function to resume fitting of models from the last successful fit during the event of a model failure. 


### Requirements

- NumPy, SciPy, Pandas
- Astropy (for astronomical coordinate transformations)
- Matplotlib (for visualization)
- scikit-image (for blob detection algorithms)
- PyYAML (for configuration management)
- 3ML and astromodels (for model generation)

### Setup

[Installation instructions](https://github.com/HAWC-Data-Release/aerie)

Installation is based on Pixi, a modern package manager built on conda-forge and PyPI. 

Lock file (pixi.lock) ensures exact reproducibility. 

Start by installing Pixi. 
```bash
curl -fsSL https://pixi.sh/install.sh | sh
```

Or download the latest release from Github. 
```bash
https://github.com/prefix-dev/pixi/releases
```

For the aerie installation, use the following procedure: 

```bash
git clone https://github.com/HAWC-Data-Release/aerie

cd aerie

# Install dependencies (downloads everything automatically)
pixi install

# Build all C++ components (185 executables)
pixi run build

# Install into the pixi environment
pixi run install

# Enter an interactive shell with aerie
pixi shell

#Install 3ML wit
pixi install -e threeml
pixi run -e threeml build 
pixi run -e threeml install

# Enter an interactive shell with aerie and 3ML
pixi shell -e threeml

# Instead of extra dependencies 
pip install scikit-image
pip install PyYAML

```
